14:T2dd3,<h1>TOEFL iBT は Home Edition がいいぞ！</h1>
<p>もう Home Edition 一択だと思うのです。てことで今回はTOEFL iBT Home Edition(自宅受験)を推しまくる記事。</p>
<p>僕はTOEFL iBTを合計8回受けましたが、最初の3回は試験会場で受験、後の5回はずっとHome Editionです。ていうのも<strong>会場受験に比べて全然体験が良かったので、Home Editionに移行しました</strong>。</p>
<p>ネットのTOEFL対策界隈を見ると、あまりHome Editionを推している記事がないです。代わりに「おすすめの受験会場ランキング」みたい記事があり、人気のところはすぐに埋まったりするそうです。そんな予約競争に労力割くよりHome Editionがいいと思います。</p>
<h1>会場受験のダメなところ</h1>
<p>挙げたらキリがないですが、体験がとにかく悪いです。</p>
<h2>他の人のSpeakingやマイクチェックがうるさい</h2>
<p>受験者が時間差でテスト開始するので、自分が一生懸命Listeningしてる時に他の人のSpeaking音声を聞かされるというバグみたいな仕様です。</p>
<p>極め付けは有名な（？）マイクチェックのための「I live in Tokyo」の合唱。受験者が次々に入室してきて順次マイクチェックのため「I live in Tokyo」と何度も繰り返します。読み上げる英文が画面に出ているのに、それを無視して「I live in Tokyo」を繰り返すのは何で？どっかの塾で教えてるんですか？</p>
<h2>会場の当たり外れ</h2>
<p>前述の通り、人気会場といわれるものがあり、そういう会場は即埋まります。僕も会場変えて受験しましたが、「人気会場」とかいってもフツーに他の人のSpeakingの音声は響き渡るので、大差ないと思います。せいぜいビルが綺麗かとかそんな違いです。</p>
<h2>待ち時間が長い</h2>
<p>集合時間の30分前とかに受付しても、すでに多くの受験者が集合していて入室→試験開始まで結構待ちます。ソラシティだと受験者数も多いので30分以上待ちます。</p>
<p>他の人のSpeakingに邪魔されないように、早く行って先に試験を始めるみたいなテクニックがあります。自分が一番先に試験開始できればListeningを誰にも邪魔されない(Writingの時に邪魔されるだろとツッコミたくなりますが)という涙ぐましい戦略で、みんな早く行くそうです。</p>
<h2>キーボードがJIS配列</h2>
<p>エンジニアの人向けですが、TOEFLの試験会場のキーボードはJIS配列です。普段MacでUSキーボード使ってると、いきなりWindowsでJISキーボード触るとテンパります。<code>'</code> や <code>"</code> を探したり、タイプミスが増えます。</p>
<p>会場によっては、ノートPCの内蔵JISキーボードでさらに癖が強いことがあります。ソラシティはUSB接続の外付けJISキーボードで幾分打ちやすいです。</p>
<h1>結論、Home Editionがオススメである</h1>
<p>会場受験のダメなところで上げた欠点が全て克服されます。</p>
<h2>Home Editionは大学に提出できるか？</h2>
<p>公式の説明によれば、 <em>「TOEFL iBT Home Edition は、テスト センターで受ける TOEFL iBT テストと同じなので、スコアは同じように受け入れられ、使用できます。」</em> と説明があります。</p>
<p>実際、Home Editionのスコアを受け付けていない大学院を<strong>見たことがない</strong>です。念の為、下記のTOEFLの公式サイトからHome Editionを受け付けている大学を確認できるので、志望校をチェックすることをオススメします。</p>
<p><a href="https://www.ets.org/toefl/test-takers/ibt/scores/understand-scores/home-edition-acceptance.html" rel="noopener noreferrer" target="_blank">https://www.ets.org/toefl/test-takers/ibt/scores/understand-scores/home-edition-acceptance.html</a></p>
<p>さらに大学のサイトでも出願要件を確認しておけば完璧だと思います。</p>
<p>ちなみに、GREも「Test at Home」という自宅受験を提供していますが、これはあまり受け付けている大学がないと思います。会場受験じゃないとダメという大学が多いのでGREは会場で受けるようにしましょう。TOEFLほど酷い体験ではありません。</p>
<h2>Home Editionのメリット</h2>
<h3>静かな自室で集中して受けられる</h3>
<p>他人のSpeaking音声に邪魔されることはありません。</p>
<h3>予約の枠が多く開始時間も柔軟である</h3>
<p>20分刻みで細かく受験開始の予約を取ることができる。直前でも予約枠は空いており、今から2週間後に受けよう！みたいな柔軟なスケジュールができる。</p>
<p>会場受験だと1ヶ月前に人気のところは埋まるし、開始時間も朝・昼・夕方くらいしか選べない。</p>
<h3>待ち時間なし</h3>
<p>時間になったらプロクター(試験官)と即接続して、試験の事前確認後に試験開始。</p>
<h3>移動時間なし</h3>
<p>受験会場まで出かけて行って受験して帰るという、1日がかりの仕事で消耗する必要なし。午後休とってサクッと受けられる。</p>
<h3>慣れた環境で受験</h3>
<p>使い慣れたキーボードでWritingに集中できます。エアコンの温度なども自分で快適に設定できますね。</p>
<h2>Home Editionのデメリット</h2>
<p>人によってはデメリットはあると思います。下記のようなものがあります。</p>
<h3>環境の用意が必要</h3>
<p>他に誰もおらず、ドアが閉まる部屋で受験することが受験要件です。家に自室がない、3時間もドアを閉めておける空間がない、常に家が騒がしいという人は難しいかもしれません。自宅に環境がない場合は、最近はコワーキングスペースにリモート会議ブースがあったり、スペースマーケット等で個室の仕事部屋を借りられるので検討してみるといいと思います。</p>
<p>それから、PCと安定したインターネット接続も必要です。PCのスペックもそれなりのメモリ容量だったと思うので、あまりに古いPCだと環境チェックをパスできないと思います。</p>
<h3>音声が若干悪い</h3>
<p>これは僕のPC環境の問題かもしれませんが、リスニング音声が若干ブツブツ切れるようなノイズが乗ってました。まあ回答には影響ないレベルですが、少し気が散るかもです。（会場受験の他人のSpeaking音声よりは全然気にならないです）</p>
<h3>プロクターとのやりとりが英語で必要</h3>
<p>試験開始前に外国人のプロクター（試験官）と環境チェックをやる必要があります。部屋の様子を見せたり、パスポートで本人確認したり、手首に何も付けてないかとか、そういう簡単なチェックの会話です。</p>
<p>チャットボックスもあって、指示がチャットで飛んできて簡単に返事したりします。（AgreeとかUnderstood程度の返事で大丈夫です）</p>
<p>人によっては全部日本語で済む会場受験が安心という人もいるかもですが、留学目指してるならこんな簡単な会話でビビってはいけないと思います。</p>
<h3>稀に試験開始前にトラブる</h3>
<p>1回だけあったのですが、試験が始められないことがあって、PCの再起動後に無事に受験できたことがあります。</p>
<p>その間、プロクターとやりとりして、さらに技術担当のプロクターに接続するといった形で、向こうも問題解決するまでちゃんと対応してくれます。</p>
<p>問題が起きても心を乱されないように心がけ、試験開始のボタンを押す前にちゃんと気持ちを切り替えて臨みましょう。</p>
<h1>準備</h1>
<p>特別な準備といえば、ペンとホワイトボードです。試験問題の漏洩を防ぐため、鉛筆と紙は使えません。</p>
<p>ペンに関してはこだわりがあって、いろいろ試した結果、下記の <a href="https://amzn.asia/d/4HlZWjE" rel="noopener noreferrer" target="_blank">バタフライボード 0.5mm極細ホワイトボードペンシル4本セット(黒)(Amazon)</a> が細くていいです。</p>
<p><img src="/assets/blog/2025_06_18/813HJI4+2NL._AC_SX679_.jpg" alt=""></p>
<p>公式のペンだと太すぎて、走り書きでメモを取ると文字が潰れてしまい、自分で何を書いたかわからなくなります。極細ペンだと小さいスペースで文字が潰れずに走り書きができます。公式の太いペンはキャップの消しゴム部分を使うためだけに使ってました。</p>
<p>ホワイトボードに関して、オススメは公式のこれです: <a href="https://www.officialtestprep.jp/c/ibt/shenote" rel="noopener noreferrer" target="_blank">TOEFL iBT(R) Home Edition 用ノートテイキング セット</a></p>
<p><img src="/assets/blog/2025_06_18/000000000375-01-xl.jpg" alt=""></p>
<p>オススメというか、特にこだわりなかったので↑を買いました。100均で揃えられそうなものですが、TOEFL受験料に比べると誤差なので普通に買いました。より快適さを求める人は工夫してみてください。</p>
<h1>対策</h1>
<p>TOEFLの勉強は前回の記事、<a href="/posts/2025_06_16_getting_100_scores_in_toefl">TOEFLiBT100点までの道のりと対策の記録</a> で紹介した中国TPOを使う人が多いと思います。</p>
<p>そこで、練習でも本番のホワイトボードとマーカーを使い、自分なりのメモの取り方を身につけましょう。</p>
<h1>当日試験前のチェック</h1>
<p>5回のHome Edition受験を通して培った僕の「試験前チェックリスト」です。プロクターに接続する前に確認しましょう。</p>
<ul>
<li>ペンは3本用意すること:
<ul>
<li>1本はキャップが消しゴムの太ペン、残りの2本は0.5mmペン</li>
<li>2本の0.5mmのうち、1本はほぼ新品のものにする</li>
</ul>
</li>
<li>ペンの書き味を確認すること:
<ul>
<li>インクの出が悪ければ新品に交換する</li>
</ul>
</li>
<li>机、テスト環境の掃除:
<ul>
<li>何1つ気を逸らすものは残さない</li>
<li>PCの画面も綺麗にする。汚れがついているとReadingで気になるので</li>
</ul>
</li>
<li>余計なアプリケーションは全て閉じること:
<ul>
<li>特にChrome, Slack, Lineなど通知が飛ぶ系</li>
<li>Macなら「おやすみモード」も使い、通知は完全OFF</li>
<li>もちろんスマホも通知OFF(フライトモードでもいい)</li>
</ul>
</li>
<li>キーボードの入力を「英数入力」にする:
<ul>
<li>「かな入力」でテストを始めない</li>
<li>MacのUSキーボード使いは英数にしてからKarabinerも終了させる</li>
</ul>
</li>
</ul>
<h1>おわりに</h1>
<p>ということで、今回はHome Editionの良さをとにかく語る回でした。</p>
